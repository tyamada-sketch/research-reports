# 競合調査レポート — 2026-03-26

## 調査対象
100Creative競合となり得る国内新規サービス（直近3ヶ月: 2025年12月〜2026年3月）

## 結果サマリー
**新規参入あり（3サービスを確認）**

直近3ヶ月において、バナー・広告クリエイティブのAI自動生成領域で複数の国内新規サービスが確認された。
特にD2C・EC事業者をターゲットとした低価格帯SaaSが台頭しており、100Creativeの競合になり得る。

## 詳細

### 1. Banavo（バナボ）
- **サービスURL**: https://banavo.net/index.html
- **提供会社**: 株式会社Lifunext
- **リリース時期**: β版 2025年11月25日 / 正規版 2026年3月予定
- **機能概要**:
  - LPのURLを入力するだけで商品情報を自動解析し、広告コピー＋バナーデザインを自動生成
  - プロンプト設計不要の3ステップ操作（商品名・LP URL・ターゲット層を入力するだけ）
  - 独自AI画像生成技術「Nanobanana Pro」を採用
  - 1回の操作で約3分以内にバナー生成完了
  - 今後：業種別テンプレート、ブランドガイドライン自動適用、CTR予測スコア、Google広告・Meta広告直接連携を計画
- **価格帯**:
  - Starter: 月額9,800円（月50本、1本あたり196円）
  - Professional: 月額29,800円（月200本、1本あたり149円）
  - Enterprise: 月額個別見積もり（無制限）
- **ターゲット**: フリーランス・中小事業者（Starter）/ 中堅広告代理店・EC企業（Professional）/ 大手代理店・大手EC企業（Enterprise）
- **導入実績**: ヘアケア、スキンケア、健康食品、不動産、飲食など15社以上がβ版利用中
- **100Creativeとの比較**:
  - 類似点: LP URLを起点にしたAI自動生成、D2C・EC事業者向け
  - 差別化の焦点: Banavoは「バナー制作コスト削減」に特化したシンプルな低価格SaaS。100Creativeはより広いSNS広告クリエイティブの多様な生成・管理機能の提供が想定される
  - 脅威度: **高** — 価格設定・ターゲット・機能範囲が100Creativeと直接競合

### 2. アド.com「バナー生成AI」機能
- **サービスURL**: https://prtimes.jp/main/html/rd/p/000000015.000125025.html
- **提供会社**: 株式会社アドドットコム
- **リリース時期**: 2025年12月（バナー生成AI機能リリース）
- **機能概要**:
  - 29億件以上の広告データを保有するSNS広告競合分析ツール「アド.com」に追加された新機能
  - 商品・サービスのURLを入力するだけでLPを自動解析し、「勝てる」バナー広告を瞬時に生成
  - 訴求軸・デザインテイストの異なる8パターン16枚を一度に生成
  - Google最新AIモデル「Nano Banana Pro」対応で日本語文字入れを高精度に実現
  - Google、Yahoo!、TikTok、Instagramなど10大メディア対応
  - 無料トライアルあり
- **価格帯**: 非公開（要問い合わせ / 無料トライアルプランあり）
- **ターゲット**: 広告代理店、制作会社、大手テクノロジー企業、アフィリエイト事業者
- **100Creativeとの比較**:
  - 類似点: URLベースのAIバナー生成、SNS広告対応のマルチメディア展開
  - 差別化の焦点: アド.comは「競合広告データ分析」と「クリエイティブ生成」を組み合わせた分析寄りのプラットフォーム。100Creativeはクリエイティブ生成の効率化・自動化に注力するSaaSとして住み分け可能
  - 脅威度: **中** — 分析ツールがメインのため直接競合はしにくいが、機能拡張次第では脅威になり得る

### 3. CREATIVE BLOOM DISPLAY Ads（博報堂DYグループ）
- **サービスURL**: https://www.hakuhodody-holdings.co.jp/news/corporate/2025/05/5441.html
- **提供会社**: 博報堂DYホールディングス / 博報堂テクノロジーズ
- **リリース時期**: 2025年5月（ディスプレイ広告制作支援）/ 2025年7月（動画分析AI追加）/ 2025年12月（RSAデッキ自動最適化機能追加）
- **機能概要**:
  - 統合マーケティングプラットフォーム「CREATIVITY ENGINE BLOOM」の一部
  - AIによるバナー広告レイアウト構成案の複数生成（構成提案生成AI）
  - 自然言語によるイメージ素材検索（素材検索AI）
  - 配信前の広告効果予測（静止画分析AI）
  - 第三者視点の印象可視化（要素解釈AI）
  - 動画素材の大量分析・評価（動画分析AIエージェント）
  - リスティング広告RSAデッキの自動最適化機能（TEXT Adsとして2025年12月追加）
- **価格帯**: 非公開（代理店向けエンタープライズ / 要問い合わせ）
- **ターゲット**: 博報堂DYグループのクライアント企業（主に大手企業）、広告代理店
- **100Creativeとの比較**:
  - 類似点: AI活用の広告クリエイティブ生成・分析
  - 差別化の焦点: 博報堂グループ向けの内製プラットフォームであり、外部SaaSとして一般販売されていない。100Creativeはオープンな月額SaaSとして中小D2C事業者・制作会社をターゲットにするため、市場セグメントが異なる
  - 脅威度: **低〜中** — 直接の市場競合ではないが、将来的に外部提供に転換した場合は強力な競合になり得る

- 2025年後半から2026年初頭にかけて、**バナー・広告クリエイティブのAI自動生成SaaS**の国内参入が加速している

- 100Creativeの差別化として「D2C事業者に特化した広告効果改善への踏み込み」「SNS広告全体の一元管理」「制作会社・代理店向けのホワイトラベル対応」等を強化することが推奨される
